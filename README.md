# Version

数据版本中心


## 项目

 - 项目名称：eProject-Version

### 项目说明

 - 语言：JAVA8
 - IDE：IDEA
 - 构建工具：Gradle 5.6
 - 框架：Netty + SpringWebFlux + Mongo
 - 接口：HTTP
 - CVS：GIT

### 项目结构
```
├─boot-version             eProject Version 服务 模块
├─core                     eProject core 核心模块
├─server                   eProject server 服务模块
├─client                   eProject client 客户端模块
```

## 功能

存储HTTP接口数据版本
